# Vernacular News Fact Checker

## 📌 Problem Statement
Pipeline Optimization – Automated Fact-Checker for Vernacular News

## 🚀 Project Overview
This project detects misinformation in regional language news by translating, extracting claims, and verifying them using AI and external sources.

## ⚙️ Features
- Multi-language support (Telugu, Hindi, Tamil, English)
- Automatic translation
- Claim extraction
- Fact-check classification (True / False / Misleading)

## 🧰 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- APIs: Translation + AI APIs

## ▶️ How to Run

```bash
git clone https://github.com/your-team/vernacular-fact-checker
cd vernacular-fact-checker
